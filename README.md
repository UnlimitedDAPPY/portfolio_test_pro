# Portfolio Automation Framework

## 📌 Project Overview
This is a Selenium WebDriver automation framework built with JavaScript, Mocha, and Chai.  
It tests the contact form functionality of a portfolio website.

---

## ⚙️ Tech Stack
- Selenium WebDriver
- JavaScript (Node.js)
- Mocha (Test Framework)
- Chai (Assertions)
- Page Object Model (POM)
